# Tab PowerToy

一款简单实用的Chrome浏览器插件，帮助您快速修改当前标签页的标题。

## 🌟 功能特点

- **深色模式支持**：自动跟随系统主题切换深/浅色界面，夜间使用更护眼
- **一键修改**：点击插件图标，输入新标题，立即生效
- **图标自定义**：支持修改网页图标(favicon)，可使用URL或emoji
- **最近标签页切换**：使用 `Alt+Q` (Windows) 或 `Option+Q` (macOS) 快捷键在最近使用的标签页之间快速切换
- **可视化面板**：
  - 覆盖全屏的现代化 UI 面板
  - 严格按最近使用顺序 (MRU) 排列
  - 支持显示 Chrome 标签页分组 (带颜色的名称胶囊)
  - 智能显示当前平台的快捷键提示
  - **快捷跳转**：每个标签页分配唯一的快捷键 (1-4, Q-V)，按住 Alt/Option + 对应键即可直达
- **简洁界面**：清晰直观的用户界面，无需复杂操作
- **键盘友好**：支持Enter键快速提交
- **即时反馈**：修改成功后有明确提示
- **自定义快捷键**：支持在Chrome扩展管理中自定义切换快捷键

## 📋 使用场景

- **工作整理**：给重要的参考页面设置更有意义的标题
- **多标签管理**：当打开多个相似页面时，设置不同标题便于区分
- **临时标记**：为需要稍后查看的页面添加提醒性标题
- **个性化体验**：根据个人喜好自定义标签页标题

## 🚀 使用方法

1. 打开您想要修改标题的网页
2. 点击Chrome工具栏中的插件图标（蓝色"T"图标）
3. 在弹出的输入框中输入新的标题
4. 点击"确定"按钮或按Enter键
5. 完成！标签页标题已更新

### 右键快捷设置

1. 在任意网页上选中一段文字
2. 点击鼠标右键打开上下文菜单
3. 选择“设为标签页标题”
4. 当前标签页标题将立即变更为选中的文字

### 最近标签页切换 (PowerToy Switcher)

1. **基本用法**：
   - 按下 `Alt + Q` (Windows) 或 `Option + Q` (Mac) 呼出面板
   - **保持按住** `Alt` / `Option` 键，面板将保持显示
   - 再次按 `Q` 键可向下选择列表中的下一个标签页
   - **松开** `Alt` / `Option` 键，立即切换到选中的标签页

2. **快捷键直达**：
   - 面板打开时，每个条目左侧显示分配的快捷键（如 `1`, `2`, `W`, `E`...）
   - **保持按住** `Alt` / `Option` 键，按下对应的字母或数字键
   - 立即切换到该标签页，无需逐个选择

3. **如需自定义快捷键**：
   - 打开Chrome扩展管理页面 (`chrome://extensions/`)
   - 点击左上角菜单，选择"键盘快捷键"
   - 找到"Tab PowerToy"扩展
   - 在"在最近使用的两个标签页之间切换"选项中设置您喜欢的快捷键 (例如 `Ctrl+Space`)
   - 插件会自动识别您的新快捷键并更新面板上的提示文案

4. **快速搜索**：
   - 在面板打开时，按下 `/` 键即可激活搜索框
   - 输入关键词快速筛选标签页
   - 支持标题/URL搜索
   - **交互细节**：
     - 按下 `/` 键：激活搜索框并聚焦
     - 搜索框激活时按下 `ESC`：退出搜索框聚焦状态，保留面板
     - 再次按下 `ESC`：关闭面板

### 修改网页图标(Favicon)

1. 在插件弹窗中，找到"输入图标URL或从下方选择emoji"输入框
2. 输入一个有效的图片URL，或从下方emoji列表中选择一个表情
3. 点击"修改图标"按钮
4. 完成！网页图标已更新

## ⚙️ 安装方法

### 从Chrome网上应用店安装（即将上线）

1. 访问Chrome网上应用店
2. 搜索"Tab PowerToy"
3. 点击"添加到Chrome"按钮

### 开发者模式安装

1. 下载此仓库的ZIP文件并解压，或使用git克隆
2. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含此插件的文件夹
6. 插件图标将出现在Chrome工具栏中

## 🔧 故障排除

### 快捷键失效问题

如果发现快捷键（Alt+Q）不起作用，请尝试以下解决方案：

1. **检查快捷键冲突**
   - 打开Chrome扩展管理页面 (`chrome://extensions/`)
   - 点击左上角菜单 → "键盘快捷键"
   - 确认没有其他扩展占用相同的快捷键

2. **重新加载扩展**
   - 打开Chrome扩展管理页面
  - 找到"Tab PowerToy"扩展
   - 点击"重新加载"按钮

3. **查看调试日志**
   - 点击浏览器右上角的插件图标打开 Popup
   - 点击底部的 **"Copy Debug Logs"** 按钮
   - 将日志粘贴到文本编辑器中查看详细的运行记录 (包含按键事件、切换逻辑等)

4. **调试模式 (旧版)**
   - 在 Popup 中点击 "Debug Open Panel" 可以强制打开面板进行测试

### 标签页切换不工作

1. **确保有足够标签页**
   - 至少需要打开2个普通网页（非chrome://页面）
   - 在多个标签页之间切换几次建立历史记录

2. **检查标签页类型**
   - 插件会自动忽略chrome://、edge://等内部页面
   - 只在普通网页之间进行切换

3. **清除历史记录**
   - 使用调试面板的"Clear History"按钮
   - 或重新加载扩展重新开始
